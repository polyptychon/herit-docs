#### 1. Must use HTTPS

#### 2. Must use headers

* Accept: "application/json" (http://dev.desk.com/API/using-the-api/#general)
* Authorization: "Basic ..." (HTTP Basic Auth) (http://dev.desk.com/API/using-the-api/#authentication)
